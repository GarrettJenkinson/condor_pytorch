condor_negloglikeloss

condor_negloglikeloss(logits, labels, reduction='mean')

computes the negative log likelihood loss described in

condor tbd.

parameters

  • logits : torch.tensor, shape(num_examples, num_classes-1)

    outputs of the condor layer.

  • labels : torch.tensor, shape(num_examples, num_classes-1)

    true labels represented as extended binary vectors (via condor_pytorch.dataset.levels_from_labelbatch).

  • reduction : str or none (default='mean')

    if 'mean' or 'sum', returns the averaged or summed loss value across all data points (rows) in logits. if none, returns a vector of shape (num_examples,)

returns

  • loss : torch.tensor

    a torch.tensor containing a single loss value (if reduction='mean' or 'sum') or a loss value for each data record (if reduction=none).

examples

    >>> import torch
    >>> labels = torch.tensor(
    ...    [[1., 1., 0., 0.],
    ...     [1., 0., 0., 0.],
    ...    [1., 1., 1., 1.]])
    >>> logits = torch.tensor(
    ...    [[2.1, 1.8, -2.1, -1.8],
    ...     [1.9, -1., -1.5, -1.3],
    ...     [1.9, 1.8, 1.7, 1.6]])
    >>> condor_negloglikeloss(logits, labels)
    tensor(0.4936)
